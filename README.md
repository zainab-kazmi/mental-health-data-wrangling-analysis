# Mental Health Data Wrangling Analysis

Python data wrangling project analyzing the relationship between workplace mental health culture and national suicide rates using global datasets. The project demonstrates data cleaning, exploration, and visualization skills applied to real-world public health and workplace data.

## Datasets
- Global suicide rates by country (publicly available)
- Workplace mental health survey data

## Project Goals
- Explore patterns and correlations between workplace mental health initiatives and national suicide rates
- Identify key insights to inform mental health awareness and policy
- Demonstrate data wrangling, cleaning, and visualization techniques in Python

## How to Run
1. Clone this repository:  
   ```bash
   git clone https://github.com/zainab-kazmi/mental-health-data-wrangling-analysis.git

2. Open mental_health_data_wrangling_analysis.ipynb in Jupyter Notebook or VS Code with the Jupyter extension

3. Run the notebook cells to reproduce the analysis

Key Skills Demonstrated

- Data cleaning and preprocessing with Python (pandas, numpy)

- Exploratory data analysis and visualization (matplotlib, seaborn)

- Report generation and summarization of insights

- Working with multiple datasets and merging data for analysis

mental_health_data_wrangling_report.pdf – a concise report summarizing analysis, findings, and visualizations
